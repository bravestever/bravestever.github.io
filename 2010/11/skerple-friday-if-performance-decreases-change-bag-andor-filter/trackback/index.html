Skerple Friday: If performance decreases, change bag and/or filter

By Katt | November 26, 2010

Look at that, the scanner’s back up! And the last Skerple is dying.

I let my Sketchbook Project sketchbook sit untouched for about a month because I was really intimidated by having it look perfect. That’s kind of crazy, because it’s a sketchbook. So I’m just penciling stuff in and after that’s done I’ll go back and make it look like something, perhaps. I’ve realized that if I’m ever going to draw any sort of scene, I’m going to have to learn to draw scenery. I’ve always hated drawing backgrounds. Maybe I can make a living off drawing floating disembodied faces, because I’m not so bad at those.
